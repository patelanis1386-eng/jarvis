## Quick Start

### Prerequisites

- Python 3.12+
- Node.js 20+
- PostgreSQL 16+
- Redis 7+
- Docker & Docker Compose (optional but recommended)
- Git

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-org/jarvis-x.git
cd jarvis-x

# 2. Backend setup
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\Activate.ps1
cd backend
pip install -r requirements.txt

# 3. Configure environment
cp .env.example .env
# Edit .env with your database credentials and API keys

# 4. Run database migrations
alembic upgrade head

# 5. Seed default data
python scripts/seed.py

# 6. Start backend
uvicorn app.main:app --reload --port 8000

# 7. Frontend setup (in a new terminal)
cd frontend
npm install
cp .env.example .env.local
npm run dev
```

The frontend will be available at `http://localhost:3000` and the backend API at `http://localhost:8000`. API docs are at `http://localhost:8000/docs`.

### Docker (one-command setup)

```bash
docker compose up --build
```
